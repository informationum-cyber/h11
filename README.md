# Hansel Eleven

Hansel Eleven is a transformation and professional enablement company focused on hybrid project delivery, agile transformation, mentorship, and modern workforce development.

This repository contains the front-end application for Hansel Eleven, built with modern web technologies and deployed on Netlify.

## Tech Stack
- **Framework:** TanStack Start
- **Frontend:** React 19, TanStack Router v1
- **Styling:** Tailwind CSS 4
- **Language:** TypeScript 5.7
- **Deployment:** Netlify

## Getting Started

To run this project locally, you will need Node.js installed.

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```
   Or use the Netlify CLI for local emulation:
   ```bash
   netlify dev
   ```

3. **Open the app:**
   Navigate to `http://localhost:8888` (if using Netlify CLI) or `http://localhost:3000` (if using standard Vite dev server) in your browser.
